# Week2 Mini Proj Demo - Deploy Marco polo on AWS Lambda
I deployed the Marco Polo Rust command line tool on AWS lambda

## Usage
Some commands to run include
* `make format`
* `make lint`
* `make run`
* `make release-arm`
* `make deploy`
* `make invoke`
